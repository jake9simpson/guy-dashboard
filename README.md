# The Guy Report

A live gold and silver dashboard showing real-time prices, historical charts, technical analysis, and mining stock performance.

## Features

Live price cards that update every 2 seconds. Historical price charts spanning back to 2000 with candlestick, line, and area views. Bull Market Support Band overlay (20-week SMA + 21-week EMA). Gold/silver ratio tracking. Technical indicators including RSI, MACD, momentum, and volatility. Moving averages performance comparison. Mining stocks tracker (GDX, GDXJ, NEM, GOLD, AEM, WPM). Unit converter for troy ounces, grams, and kilograms.

## Tech Stack

Next.js 16, React, TypeScript, TanStack Query, lightweight-charts, Tailwind CSS, Zustand. Data sourced from Yahoo Finance.

## Getting Started

Clone the repository and install dependencies.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
